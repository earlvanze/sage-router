What this is

A qualification path for teams that want a simpler commercial relationship while Sage Router keeps model traffic authenticated, metered, revocable, rate-limited, and abuse-controlled.

Default public launch

Public Lite, Pro, and Max plans sell routing infrastructure, generated keys, quotas, analytics, reliability tooling, and support. Provider access stays customer-authorized by default.

Managed access boundary

Managed provider access is not active in public plans. It requires explicit provider authorization, published terms, positive unit economics, durable quotas, generated-key revocation, rate limits, billing controls, durable operator audit events, abuse controls, and acceptable-use enforcement.

One-subscription demand capture

Ollama, OpenAI, and Anthropic are target provider families for private-beta authorization review, not enabled public resale. Tell us which family and commercial model you would buy first.
