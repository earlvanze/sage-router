What this is

A qualification path for teams that want a simpler commercial relationship while Sage Router keeps model traffic authenticated, metered, revocable, rate-limited, and abuse-controlled.

Default public launch

Public Lite, Pro, and Max plans sell routing infrastructure, generated keys, quotas, analytics, reliability tooling, and support. Provider access stays customer-authorized by default.

Managed access boundary

Managed provider access is not active in public plans. It requires explicit provider authorization, published terms, provider terms acknowledgment, an authorized provider allowlist, positive unit economics, durable quotas, generated-key revocation, rate limits, billing controls, durable operator audit events, abuse controls, and acceptable-use enforcement.

Readiness checkpoint

The public pricing checkpoint keeps managed access disabled until a configured provider cost model and plan-margin checks for Lite, Pro, and Max pass. It can show missing controls and margin status, but it does not publish actual provider costs, provider credentials, prompts, or raw responses.
