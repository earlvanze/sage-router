Managed provider access

A guarded path toward one subscription for bundled model access.

Managed provider access is not active in the public plans yet. It will stay disabled until Sage Router has explicit provider resale terms, a published margin policy, durable quotas and rate limits, durable operator audit events, and acceptable-use terms for managed model access.

  • Default launch: customer-authorized providers and local models.
  • OpenRouter remains supported for customer-authorized BYOK routing, not bundled public resale.
  • OpenRouter-specific buyers can review the Sage Router vs OpenRouter page before choosing a hosted marketplace API, local-first routing layer, or both.
  • Future path: private-beta managed access after contract, billing, and abuse-control readiness checks pass.
  • Readiness is exposed in the public /pricing metadata as managedProviderAccess.
  • Positive unit economics must be backed by a configured cost model and plan-margin checks before bundled model access can become active.
  • Prerequisite boundaries are published as provider resale terms, a margin policy, and acceptable use.
